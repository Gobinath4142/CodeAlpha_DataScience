# CodeAlpha_DATASCIENCE
# CodeAlpha Internship Projects – Data Science

This repository contains my solutions for *CodeAlpha’s Data Science Internship projects*. The projects focus on data analysis, exploratory data analysis (EDA), data visualization, and predictive analytics using Python.
---

## 📂 Tasks Overview



### ✅ Task 1: Iris Flower Classification



* Loaded and analyzed the Iris dataset.

* Performed data preprocessing and exploratory data analysis.

* Built a classification model to identify Iris flower species.

* Evaluated model performance using standard metrics.

* **Libraries used:** pandas, scikit-learn, matplotlib.



### ✅ Task 2: Unempolyment Analysis



* Analyzed student academic performance data.

* Explored relationships between study habits and exam scores.

* Created visualizations to identify trends and patterns.

* **Libraries used:** pandas, matplotlib.



### ✅ Task 3: Car Price Prediction with Machine Learning



* Developed a machine learning model to predict car prices.

* Performed feature engineering and data preprocessing.

* Trained and evaluated a regression model for accurate predictions.

* **Libraries used:** pandas, scikit-learn, matplotlib.



### ✅ Task 4: Sales Prediction using Python



* Built a sales forecasting model using advertising data.

* Analyzed the impact of TV, Radio, and Newspaper advertisements.

* Used regression techniques to predict future sales.

* **Libraries used:** pandas, scikit-learn, matplotlib.
